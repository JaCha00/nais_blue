## Default Permission

Allows the NAIS UI to control bounded Android R2 and paired-LAN transfers.

#### This default permission set includes the following:

- `allow-schedule`
- `allow-pause`
- `allow-resume`
- `allow-cancel`
- `allow-retry`
- `allow-checkpoint`
- `allow-status`
- `allow-recover`
- `allow-configure-cloudflare`
